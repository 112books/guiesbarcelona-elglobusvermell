1. Accés a l'editor

L'editor funciona amb un token personal de GitHub (PAT). És una clau d'un sol ús que identifica el teu usuari sense necessitat de contrasenya.

Primera vegada: crear el token

  1. Entra a GitHub → Settings → Fine-grained tokens.
  2. Posa un nom descriptiu, per exemple Editor Guies Barcelona.
  3. Tria una caducitat d'1 any.
  4. A Repository access, selecciona Only select repositories i afegeix 112books/guiesbarcelona-elglobusvermell.
  5. A Permissions → Repository permissions, activa:
    • Contents: Read and write
    • Metadata: Read (s'activa sol)
  6. Clica Generate token i copia el token (apareix una sola vegada).
Desa el token en un lloc segur (gestor de contrasenyes o nota privada). Si el perds hauràs de crear-ne un de nou seguint els mateixos passos.

Entrar a l'editor

  1. Ves a guiesbarcelona.elglobusvermell.org/admin/cms/.
  2. Enganxa el token al camp i clica Sign in using Access Token.
  3. L'editor carrega la llista d'edificis.

2. La llista d'edificis

Un cop dins l'editor, veuràs la col·lecció Edificis i elements. Conté tots els edificis del mapa, organitzats per publicació.

Filtrar per publicació

Al menú de la dreta (View) pots agrupar els edificis per guia: GATCPAC, Arquitectura Moderna, La Barceloneta, etc. Cada edifici apareix en tants grups com guies té assignades.

Filtrar per estat

Al mateix menú pots filtrar per Publicats (visibles al web) o Esborranys (ocults).

Cercar

Fes servir el camp de cerca a la barra superior per trobar un edifici per nom o adreça.

3. Editar un edifici

Clica el nom d'un edifici per obrir-ne la fitxa. Modifica els camps que calgui i desa els canvis.

Els canvis es guarden directament al repositori de GitHub. En 1-2 minuts el web s'actualitza automàticament.

4. Descripció dels camps

Camp Descripció
Títol obligatori Nom oficial de l'edifici. Apareix al mapa i a la fitxa.
Esborrany Si està activat, l'edifici no apareix al web. Útil per preparar fitxes incompletes.
Publicacions opcional Guia(es) on apareix l'edifici. Un edifici pot estar a diverses guies alhora.
Temes transversals opcional Categories que creuen publicacions: Espai públic, Art públic, Dones arquitectes, Nomenclàtor femení.
Adreça opcional Adreça postal completa. Exemple: Carrer de Provença, 261.
Any de construcció opcional Any o període. Exemples: 1934, c. 1930, 1928-1931.
Arquitectes opcional Un nom per línia. Exemple: Josep Lluís Sert.
Tipologia opcional Exemple: Habitatge col·lectiu, Equipament cultural, Espai públic.
Nivell de protecció opcional Del Catàleg del Patrimoni Arquitectònic de Barcelona. Nivells A (BCIN) fins a D (Documental).
Superfície (m²) opcional Superfície construïda en metres quadrats (número enter).
Anys de reforma opcional Un any per línia, per a reformes o rehabilitacions significatives.
Premis opcional Un premi per línia. Exemple: Premi FAD 1998.
Latitud / Longitud opcional Coordenades GPS per situar l'edifici al mapa. Exemples: 41.3851 / 2.1734. Sense coordenades, l'edifici no apareix al mapa.
Descripció opcional Text descriptiu principal de la fitxa. Text pla, sense format.
Text nomenclàtor femení opcional Només per als elements de la categoria Nomenclàtor femení.
Contingut addicional opcional Text enriquit (negreta, llistes, enllaços). Apareix a continuació de la descripció.

5. Editar les pàgines del web

A la barra lateral esquerra de l'editor, sota Pàgines del web, trobaràs:

  • Presentació: text de la pàgina principal (inici del web).
  • En paper: descripció de les publicacions físiques.
  • Crèdits: autors, col·laboradors i agraïments.

Les pàgines s'editen amb un editor de text enriquit (negreta, llistes, enllaços, etc.).

6. Desar i publicar

  1. Modifica els camps que calgui.
  2. Clica el botó Save (a la part superior dreta).
  3. El canvi es puja directament a GitHub. En 1-2 minuts el web reflecteix la modificació.
No cal fer cap pas addicional: desar equival a publicar. Si vols preparar una fitxa sense publicar-la, activa el camp Esborrany.

7. Que NO fer

No creïs edificis nous des d'aquest editor. La creació de nous elements requereix coordinació tècnica.
No eliminis edificis. Si un element ha de desaparèixer del web, activa el camp Esborrany en lloc de suprimir-lo.
  • No canviïs el nom del fitxer (slug) d'un edifici: trencaries els enllaços existents.
  • No toquis els camps de coordenades si no estàs segur/a dels valors correctes.
  • No posis format markdown (asterisc, guions, corxets) al camp Descripció: utilitza el camp Contingut addicional per a text enriquit.
